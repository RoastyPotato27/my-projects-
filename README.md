## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/RoastyPotato27/my-projects-
cd my-projects-
```

### 2. Run a script

Make sure you have Python installed, then run any file:

```bash
python filename.py
```

Example:

```bash
python Calculator.py
```
